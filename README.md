# CRUD de Clientes

Este projeto oferece um CRUD (Create, Read, Update, Delete) simples para gerenciar clientes.

## Funcionalidades

- **Cadastro de clientes**
- **Listagem de clientes**
- **Busca de clientes**
- **Atualização de clientes**
- **Exclusão de clientes**
